This is yet another forked repository of FreeMind from sourceforge.net.

The original FreeMind repository is https://sourceforge.net/p/freemind/code/ci/1.1.0/tree/.

Based on the original FreeMind, this product is named **FreeMind Lite** because the following features have been removed:

- Applet plugin
- Export PDF plugin
- Flash plugin
- Map plugin
- Script plugin
- Socket plugin
- Server/Client TCP communication and HTTP proxy settings

This repository comes from https://sourceforge.net/u/nbtkyng/freemind/ci/main/tree/, which was forked from the origial.
